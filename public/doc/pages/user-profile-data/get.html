プロパティ名 値 説明
name string ユーザー名
profileImage string プロフィール画像のURL
bio string 自己紹介文
groups[] list フォローしているグループのリスト
groups[].name string グループの名前
groups[].groupId string グループID
groups[].icon string グループのプロフィール画像のURL
tags[] list タグのリスト
tags[].name string タグ名
tags[].tagId string タグID
basicInfo object 基本情報のオブジェクト
basicInfo.birthday[] list 誕生日
basicInfo.birthday[ 0 ] string 年
basicInfo.birthday[ 1 ] string 月
basicInfo.birthday[ 2 ] string 日
basicInfo.gender string 設定されている言語で表記された性別

例えば日本語に設定されている場合

  • "男" - 男
  • "女" - 女
basicInfo.genderVal string 英語で表記された性別
  • "male" - 男
  • "female" - 女
basicInfo.belongGroups[] list 所属しているグループのリスト
basicInfo.belongGroups[].name string グループの名前
basicInfo.belongGroups[].groupId string グループID
basicInfo.belongGroups[].icon string グループのプロフィール画像のURL
contactInfo object コンタクト情報のオブジェクト
contactInfo.mails[] list メールアドレスのリスト
contactInfo.mails[ n ] string メールアドレス
contactInfo.webistes[] list ウェブサイトURLのリスト
contactInfo.websites[ n ] string ウェブサイトURL
privacy object プライバシーのオブジェクト
privacy.bio string 自己紹介文の公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.gruops string フォローしているグループの公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.tags string タグの公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.gender string 性別の公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.birthday string 誕生日の公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.belongGroups string 所属しているグループの公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.mails string メールアドレスの公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
privacy.websites string ウェブサイトURLの公開設定
  • "private" - 非公開
  • "limited" - グループメンバーのみ公開
  • "public" - 公開
valid boolean 有効なユーザーかどうかの判別
  • true - 有効
  • false - 無効
permission boolean 基本情報の設定が可能かどうかの判別
  • true - 可
  • false - 不可
