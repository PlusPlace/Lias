レスポンス(JSON)

[{
    "name": {string},
    "profileImage": {string},
    "summary": {string},
    "member": [
        {
            "name": {string},
            "icon": {string},
            "userId": {string} 
        }
    ],
    "groups": [
        {
            "name": {string},
            "icon": {string},
            "groupId": {string}
        }
    ],
    "tags": [
        {
            "name": {string},
            "tagId": {string}
        }
    ],
    "basicInfo": {
        "founded": {string}
    },
    "contactInfo": {
        "mails": [
            {string}
        ],
        "websites": [
            {string}
        ]
    },
    valid: {boolean},
    permission: {boolean}
}]
プロパティ名 値 説明
name string グループ名
profileImage string プロフィール画像のURL
summary string 概要
member[] list 所属しているメンバーのリスト
member[].name string メンバーの名前
member[].icon string メンバーのプロフィール画像のURL
member[].userId string メンバーのユーザーID
groups[] list フォローしているグループのリスト
groups[].name string グループの名前
groups[].icon string グループのプロフィール画像のURL
groups[].groupId string グループID
tags[] list タグのリスト
tags[].name string タグの名前
tags[].tagId string タグID
basicInfo object 基本情報のオブジェクト
basicInfo.founded string 設立年月日
contactInfo object コンタクト情報のオブジェクト
contactInfo.mails[] list メールアドレスのリスト
contactInfo.mails[ n ] string メールアドレス
contactInfo.websites[] list ウェブサイトURLのリスト
contactInfo.websites[ n ] string ウェブサイトURL
valid boolean 有効なグループかどうかの判別
  • true - 有効
  • false - 無効
permission boolean 基本情報の設定が可能かどうかの判別
  • true - 可
  • false - 不可
