まず始めに

仕様書の作成にはHTMLおよびCSSの知識が必要です。
このテンプレートは.html, .css, .pngの静的ファイルで構成されています。 HTML5に準拠したドキュメントです。

ディレクトリ構成

  • documentation
    • pages
      • ページファイル
    • style
      • images
        • イメージファイル
      • page.css
      • template.css
    • index.html

page.css: ページファイル内のHTMLドキュメントに適用されるレイアウト情報です。
template.css: テンプレートに適用されるレイアウト情報です。
index.html: 基本テンプレートです。

サイドバーのパーツ

項目

左のサイドバーにはそれぞれ独立した項目が表示されています。
これらのパーツは以下のタグで構成されています（値はデフォルトです）。 

<div class="list-box">
    <ul class="list">
        <li class="list-title">タイトル</li>
        <li class="list-contents">
            <ul>
                <li><a href="pages/XXX/a.html" target="main-frame">リスト1</a></li>
                <li><a href="pages/XXX/b.html" target="main-frame">リスト2</a></li>
                <li><a href="pages/XXX/c.html" target="main-frame">リスト3</a></li>
            </ul>
        </li>
    </ul>
</div>

これを、ディレクトリ構成にあるindex.htmlに追加します。 追加方法は、 

<!-- セクション始め -->
...
<!-- セクション終了 -->

と表記されているタグを参考にしてください。

ページのパーツ

完成形はページの作成にあります。完成形を見ながら次の説明を読んでいくとわかりやすいかもしれません。

ヘッダ

それぞれのページの上部に表示されるパーツです。タイトル、目次があります。
これらのパーツは以下のタグで構成されています（値はデフォルトです）。 

<header>
    <h2>タイトル</h2>
    <hr />
    <!-- ここにはリストが挿入されます。 -->
    <hr />
</header>

リスト

ヘッダの目次に使用されます。
これらのパーツは以下のタグで構成されています（値はデフォルトです）。 

<ul class="list">
    <li class="list-title"><a href="#ABC">アーティクルのタイトル</a></li>
    <li class="list-contents">
        <ul>
            <li><a href="#ABC-DEF">セクションのタイトル</a></li>
        </ul>
    </li>
</ul>

ul, liをさらに入れ子にすると、 

<ul class="list">
    <li class="list-title"><a href="#ABC">アーティクルのタイトル</a></li>
    <li class="list-contents">
        <ul>
            <li class="list-contents">
                <ul>
                    <li><a href="#ABC-DEF">セクションのタイトル</a></li>
                </ul>
            </li>
        </ul>
    </li>
</ul>

となります。
リストは、ヘッダに挿入してください。

a要素の親要素がlist-titleをclass属性の値にとるli要素のとき、このa要素はハッシュでアーティクルのIDを、 それ以外のa要素はセクションのIDを参照してください。

アーティクル

独立した記事を表記します。
これらのパーツは以下のタグで構成されています（値はデフォルトです）。 

<article id="ABC">
    <h4>アーティクルのタイトル</h4>
    <p>アーティクル内の記事</p>
    <!-- 必要に応じてセクションが挿入されます。 -->
</article>

id属性の値はリストが参照するハッシュと同名にする必要があります。

セクション

アーティクル内に配置されます。ひとまとまりの記事を表記します。
これらのパーツは以下のタグで構成されています（値はデフォルトです）。 

<section id="ABC-DEF">
    <h4>セクションのタイトル</h4>
    <p>セクション内の記事</p>
</section>

コード

ソースコードを表示する際に用いられます。
これらのパーツは以下のタグで構成されています（値はデフォルトです）。 

<pre>
<code>alert( 'Hello, world' );</code>
</pre>

ページの作成

雛形を使う

ページの完成形は以下のようになります。 

<!DOCTYPE html>

<html lang="ja">

    <head>
        <title>Lias Documentation | タイトル</title>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
        <meta http-equiv="Content-Style-Type" content="text/css" />
        <link rel="stylesheet" type="text/css" href="../../style/page.css" />
    </head>
    
    <body>
        <div class="container">
            <header>
                <h2>タイトル</h2>
                <hr />
                <ul class="list">
                    <li class="list-title"><a href="#ABC">アーティクルのタイトル</a></li>
                    <li class="list-contents">
                        <ul>
                            <li><a href="#ABC-DEF">セクションのタイトル</a></li>
                        </ul>
                    </li>
                </ul>
                <hr />
            </header>
            
            <article id="ABC">
                <h4>アーティクルのタイトル</h4>
                <p>アーティクル内の記事</p>
                
                <section id="ABC-DEF">
                    <h4>セクションのタイトル</h4>
                    <p>セクション内の記事</p>
                    <pre>
<code>alert( 'Hello, world' );</code>
                    </pre>
                </section>
            </article>
        </div>
    </body>

</html>

配置について

ディレクトリ構成のpagesに特定のフォルダを作成し、そこに作成したページを配置してください。 また、サイドバーに新たな項目を追加するごとにフォルダを作成してください。
例えば、「Sample」という新たな項目をサイドバーに追加し、api.htmlを作成するとき、ディレクトリ構成は次のようになります。

  • documentation
    • pages
      • sample
        • api.html
    • style
      • images
        • イメージファイル
      • page.css
      • template.css
    • index.html
