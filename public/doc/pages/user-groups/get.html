Group API （ユーザー用） - GET

  • リクエストURL
  • パラメータ
  • レスポンス(JSON)

リクエストURL

GET http://dev-lias.plus-place.org/-/group-update.php

パラメータ

必須パラメータ 値 説明
オプションパラメータ 値 説明

レスポンス(JSON)

[{
    "group": {
        "name": {string},
        "groupId": {string},
        "icon": {string}
    },
    "updates": [
        {
            "updatedAt": [
                {number},
                {number},
                {number}
            ],
            "update": {string}
        }
    ]
}]
プロパティ名 値 説明
group object グループ情報のオブジェクト
group.name string グループ名
group.groupId string グループのID
group.icon string グループのプロフィール画像
updates[] list グループの近況
updates[].updatedAt[] list 何らかのアップデートをした日
updates[].updatedAt[ 0 ] number 年
updates[].updatedAt[ 1 ] number 月
updates[].updatedAt[ 2 ] number 日
updates[].update string 何らかのアップデートに関する情報（例えば、イベントを作成した通知だったり、誰かをメンバーに加えたという通知だったり...）

複数のグループおよび近況の定義について

複数のグループ、近況をJSONに定義する場合、次のようにしてください。 

[{
    "group": {
        ...
    }
    "updates": [
        {
            "updatedAt": [
                ...
        },
        {
            "updatedAt": [
                ...
        }
    ]
},
{
    "group": {
        ...
    }
    "updates": [
        {
            "updatedAt": [
                ...
        },
        {
            "updatedAt": [
                ...
        }
    ]
}]

ただし、アップデート情報は3個までにしてください。
